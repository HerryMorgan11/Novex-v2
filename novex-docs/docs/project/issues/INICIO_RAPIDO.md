# ⚡ INICIO RÁPIDO - Crear Issues en GitHub

## Las issues NO están en GitHub todavía

Estos archivos `.md` son plantillas. Debes crearlas tú en GitHub.

## Método más rápido (30 segundos por issue)

### 1. Abre este archivo:

```
issue-1.1-estructura-clean-architecture.md
```

### 2. Copia TODO (Ctrl+A, Ctrl+C)

### 3. Ve a:

https://github.com/HerryMorgan11/Novex-v2/issues/new

### 4. Pega (Ctrl+V)

### 5. Click "Submit new issue"

## ✅ ¡Listo!

Ahora verás la issue en:
https://github.com/HerryMorgan11/Novex-v2/issues
